# RUN Structure

Each hypothesis run follows a standard archive structure:

```text
RUN_DIR/
  input/
    hypothesis.md
    attachments/
  run.md
  outputs/
    role_outputs/
    hypothesis_summary.md
    validation_questions.md
    market_analysis.md
    hypothesis_map.md
    hypothesis_digest.txt
    customer_discovery_plan.md
    decision_review.md
    *.marker
```

`RUN_DIR` naming pattern: `runs/HYP-YYYY-MM-DD-NNN/`

---

## Principle

Each run must be:

* isolated
* reproducible
* traceable
