# Структура RUN_DIR

Каждый прогон гипотезы следует стандартной структуре:

```text
RUN_DIR/
  hypothesis.md
  run.md
  outputs/
    role_outputs/
    hypothesis_summary.md
    market_analysis.md
    hypothesis_map.md
    hypothesis_digest.txt
```

---

## Принцип

Каждый прогон должен быть:

* изолированным
* воспроизводимым
* трассируемым
